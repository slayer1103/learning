Day 2 we learnt File system and its commands such as:

Mkdir - used for creating directories
cp - for copying files
mv - moving files
head - for reading top 10 lines of a file
tail - for reading bottom 10 lines of a file
cat - for reading the file
find - using it for finding file in directories, it also can be used to find specific file types

Used touch for creating files in different directories.
